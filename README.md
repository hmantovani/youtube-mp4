### Description
A modern desktop utility for downloading YouTube videos in high quality. This program supports single or bulk downloads, with features to import lists via text files, CSV spreadsheets, or by simply pasting multiple links from the clipboard.

### Features
* Clean and responsive UI with Dark Mode support.
* Bulk download dozens of links at once by pasting in the text area.
* Dedicated import buttons to load `.txt` or `.csv` files.
* Automatically stores in a `/downloads` folder within the script directory.

### How to Run (Windows)
* Make sure you have **Python 3.x** installed and added to your PATH.
* Download or clone this repository.
* Double-click the **`run.bat`** file.
* The script will set up a virtual environment, install dependencies, and launch the app.

### Structure

```text

├── main.py
├── requirements.txt
├── run.bat
├── assets/
└── downloads/
```

# <img src="assets/br.png" width="24"> <img src="assets/pt.png" width="24"> YouTube Multi-Downloader

### Descrição
Um utilitário de desktop moderno para baixar vídeos do YouTube em alta qualidade. Este programa permite downloads individuais ou em massa, com suporte para importação de listas via arquivos de texto, planilhas CSV ou simplesmente colando múltiplos links da área de transferência.

### Funcionalidades
* Interface moderna com visual limpo e responsivo com suporte a Dark Mode.
* Download em massa de dezenas de links de uma vez ao colar no campo de texto.
* Botões de importação para carregar arquivos `.txt` ou `.csv`.
* Armazena automaticamente na pasta `/downloads` no diretório do script.

### Como Rodar (Windows)
* Certifique-se de ter o **Python 3.x** instalado e adicionado ao PATH.
* Baixe ou clone este repositório.
* Clique duas vezes no arquivo **`run.bat`**.
* O script irá configurar o ambiente virtual, instalar as dependências e abrir o programa.

### Estrutura

```text
/youtube-mp4
├── main.py
├── requirements.txt
├── run.bat
├── assets/
```
